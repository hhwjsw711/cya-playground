# TaskFlow

A team task management app built with [Convex](https://convex.dev/), React, Vite, Tailwind CSS, and Convex Auth.

## Features

- Projects with role-based membership (admin, editor, viewer)
- Kanban task board with status and priority tracking
- Task assignment, comments, and activity log
- Labels for task categorisation
- My Tasks dashboard view across all projects
- Full-text task search within projects

## Tech stack

- [Convex](https://convex.dev/) - backend, database, real-time queries
- [React 19](https://react.dev/) + [Vite](https://vitejs.dev/) - frontend
- [Tailwind CSS v4](https://tailwindcss.com/) - styling
- [Convex Auth](https://labs.convex.dev/auth) - password-based authentication

## Development

```
npm install
npm run dev
```
